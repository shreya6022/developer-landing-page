# Developer Landing Page

This is a simple developer landing page created using HTML and CSS. The project focuses on building a clean and responsive landing page layout.

## Features

- Clean user interface
- Responsive layout
- Simple navigation
- HTML and CSS only

## Technologies Used

- HTML5
- CSS3

## How to Run

1. Clone this repository

```bash
git clone https://github.com/shreya6022/developer-landing-page.git
```

2. Open the project folder.

3. Open `index.html` in your browser.
